NgRX Demo

Shown below are two parts of the view that rely on the state from our ngrx store. Click the "Load Something" button to trigger an action via the DemoService which will update the store's isLoading state to be true. After a short timeout (to simulate a server call), the DemoService dispatches another action to set it back to false. The two items below illustrate two different ways of displaying data from the store. See the demo.component.ts file.
